# Security Policy

## Reporting a Vulnerability

Please report suspected vulnerabilities responsibly.

- Open a private security advisory on GitHub if available.
- If that is not available, open an issue and clearly mark it as a security concern without sharing exploit details.

Repository issues: `https://github.com/Pallepadehat/MarkdownEditor/issues`

## Response Process

- We will acknowledge valid reports as quickly as possible.
- We will investigate and prioritize fixes based on severity and impact.
- We will publish remediation details after a fix is available.
